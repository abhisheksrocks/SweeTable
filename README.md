# Follow Along / Contribute

Clone the project

```bash
  git clone https://github.com/abhisheksrocks/SweeTable.git
```

Go to the project directory

```bash
  cd SweeTable
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
